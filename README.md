# Bookathlon

## bookathlon-backend

Backend Java sviluppato con Spring Boot per la piattaforma Bookathlon.

### Tecnologie utilizzate

- Java 21  
- Spring Boot 4.x  
- Spring Web  
- Spring Data JPA  
- Spring Security  
- Thymeleaf (solo per prototipi HTML)  
- MySQL  
- Bean Validation (Hibernate Validator)
